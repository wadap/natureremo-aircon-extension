## 拡張機能名

エアコン操作 - Nature Remo

## 短い説明（132文字以内）

Nature Remo のエアコンをブラウザからワンクリックで操作。ON/OFF・モード切替・温度調整がツールバーから素早くできます。

## 詳しい説明

Nature Remo Cloud API を使って、ブラウザのツールバーからエアコンを素早く操作できる Chrome 拡張機能です。

【できること】
・各エアコンの ON / OFF をワンクリック
・運転モードの切替（冷房・暖房・除湿・送風・自動）
・温度の調整
・複数台のエアコンをまとめて OFF

【セットアップ】
1. Nature Remo のアクセストークンを取得（https://home.nature.global/）
2. 拡張機能のポップアップにトークンを入力
3. 操作したいエアコンを選択 → すぐに使えます

【特徴】
・シンプルで直感的な操作画面
・複数台のエアコンに対応
・モード・温度の詳細設定が可能
・外部サーバーへのデータ送信なし（Nature Remo API との通信のみ）
・アナリティクス・トラッキングなし

【ご注意】
・利用には Nature Remo のアカウントとアクセストークンが必要です
・Nature Remo API のレート制限（5分あたり30回）を超えると一時的にエラーになります
